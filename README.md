# 太鼓の達人 設置店舗マップ

太鼓の達人ニジイロVerの日本国内の設置店舗マップ

**アクセス**: https://white876765.github.io/taiko_nijiiro_map/

## 概要

このサイトは、太鼓の達人ニジイロVer（以降「ニジイロ」）がプレイできるアーケード店舗を見つけるための**マップサイト**です。プレイヤーの現在地から近い店舗を検索したり、都道府県ごとに設置情報を確認したりできます。

毎日自動で店舗情報が更新されるため、常に最新の設置状況を把握できます。

## 主な機能

### マップ
- 日本全国のニジイロ設置店舗をマップ上に表示
- マップはズームイン・ズームアウトで詳細情報を確認可能
- マーカーはズームレベルに応じて自動的にクラスタリング

### 検索・フィルタリング
- **店舗名検索**：店舗名で絞り込み
- **都道府県フィルタ**：特定の都道府県の店舗のみ表示
- **台数フィルタ**：「台数3台以上」など、機械の台数で絞り込み

### 現在地検出
- ボタンをクリックすると、ブラウザの位置情報を用いて現在地をマップに表示
- 周辺の設置店舗をすぐに見つけられます

### 統計情報
- 現在表示されている店舗数と合計設置台数をリアルタイム表示

### 更新通知
- 新規設置店、店舗名変更、台数変化などを色分けマーカーで視覚的に表示
- 過去7日間の変更履歴を確認可能

## 使い方

1. **マップを見る**
   - ページを開くと、日本全国のニジイロ設置店舗がマップに表示されます

2. **店舗を検索する**
   - 上部の検索ボックスに店舗名を入力して絞り込み
   - ドロップダウンメニューから都道府県を選択
   - スライダーで必要な機械台数を指定

3. **現在地を確認する**
   - 右下の「◎」ボタンをクリックして現在地を表示
   - 周辺の設置店舗が自動的に拡大表示されます

4. **最新の変更を確認する**
   - 右上の更新通知バナーを確認
   - 過去7日間の店舗追加、削除、台数変化の履歴を確認可能

## 更新システム

このマップのデータは毎日自動で更新されます：

- **日次更新**：設置店舗データは毎日自動的に公式情報から取得・更新
- **変更トラッキング**：前日との差分を自動検出
  - 新規追加の店舗（緑マーカー）
  - 削除された店舗
  - 設置台数の増減（オレンジマーカー）
- **変更履歴**：過去7日間の更新履歴を保存し、いつ何が変わったかを追跡可能

## 注意事項

**店舗情報の正確性について**

このマップに表示される店舗名、住所、座標情報は、公式サイトから自動取得したデータを使用しています。

万が一、表示される情報に誤りや古い情報がある場合は、**公式サイト側のデータが間違っている可能性があります**。
